# Snake Game

A classic, minimalist Snake game implementation. 

## Overview
This project was built from scratch utilizing pure Python logic and Tkinter/Pygame, emphasizing core algorithmic concepts and custom game-loop management rather than relying on heavy game engines. The architecture relies on custom, object-oriented implementations for entities and grid-based rendering.

## Features
- **Pure Python Logic**: Core mechanics, including matrix movement and collision detection, are computed natively relative to the grid.
- **Object-Oriented Architecture**: Cleanly encapsulated `Snake`, `Food`, and `Board` classes.
- **Minimalist Design**: Focused entirely on raw functionality and responsive controls with a strictly technical scope.

## How to Run

### Prerequisites
- Python 3.x installed on your system.
- The `pygame` library (required for this specific version).

### Execution
1. Install the required dependency via pip:
   ```bash
   pip install pygame
   ```
2. Run the main script from your terminal:
   ```bash
   python snake_2.py
   ```
3. **Controls**: Use the arrow keys (`UP`, `DOWN`, `LEFT`, `RIGHT`) to navigate the snake and collect food. Close the application window to exit.
